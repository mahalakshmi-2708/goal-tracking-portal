# 🧠 Goal Tracking Portal (AtomQuest Hackathon)

## 🚀 Features
- Employee Goal Creation & Submission
- Manager Approval Workflow
- Quarterly Progress Tracking
- Role-based Dashboard (Employee / Manager / Admin)

## 🛠 Tech Stack
- Python
- Streamlit
- SQLite
- Pandas

## 👥 Roles
- Employee: Create & update goals
- Manager: Approve & review goals
- Admin: System control

## ▶️ Run Project
streamlit run app.py
